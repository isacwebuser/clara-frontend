# Ecclesia Frontend

Frontend implementation for the Ecclesia SaaS platform, built with Flutter.

## Getting Started

### Prerequisites

- Flutter SDK installed
- Valid device (Chrome, Edge, or Windows Desktop)

### Running the App

You can run the application using the following commands in your terminal:

**1. Run on Chrome (Web):**
```bash
flutter run -d chrome
```

**2. Run on Windows (Desktop):**
```bash
flutter run -d windows
```

### Troubleshooting

If you encounter issues with missing devices:
```bash
flutter devices
```

If the project seems broken or missing platform folders:
```bash
flutter create .
```
